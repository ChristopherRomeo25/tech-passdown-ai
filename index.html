How to get your free GitHub token (2 minutes):
1. Go to github.com/settings/tokens and click "Generate new token"
2. Give it a name like "Pass Down Agent"
3. Under Permissions, enable Models → Read (this is all it needs)
4. Click Generate token and paste it above
5. Your token stays in your browser only — it is never stored or sent anywhere except GitHub's API
