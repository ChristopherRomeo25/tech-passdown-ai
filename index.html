One-time setup — takes about 2 minutes. Your info stays in this browser only.

1
Go to GitHub → New Token (create a free GitHub account first if you don't have one)
2
Set Token name to anything (e.g. Pass Down Tool), set Expiration to 90 days or whatever you prefer
3
Under Permissions → Account permissions, find Models and set it to Read-only — this is the only permission needed
4
Click Generate token, copy it, and paste it below. Never share your token with anyone — not in Copilot, chat or email.
